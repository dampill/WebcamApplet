Java Webcam Applet Demo

Note: this page requires you to give permission for the applet to run. The applet is a peaceful creature, however, it will:

  • Contact a server to download the files required to access your webcam.
  • Store the files locally in a temporary folder./li>
  • Access your webcam and display the images on your screen (without sending it to anyone)

Please make sure that you're okay with this before starting the applet.
